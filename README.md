# Union Frères D'Assurance

Site web courtier en assurance — Casablanca, Maroc

## Structure du projet
- src/pages/UnionFreres.jsx — Page principale
- src/components/uf/ — Tous les composants Union Frères
- src/App.jsx — Routeur (route: /union-freres)
- src/index.css — Styles & tokens de design
- tailwind.config.js — Configuration Tailwind

## Installation
npm install
npm run dev

## Technologies
- React 18 + Vite
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icônes)
